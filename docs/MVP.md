# Mami Key MVP 功能清单

> 范围：仅 **家长对孩子**、**家长对老师** 两个场景。  
> 产品形态：与 LoveKey / 蜜语键盘一致——输入法插件，复制对方消息 → 选场景与风格 → 生成回复 → 粘贴发送。

---

## 一、MVP 目标

| 目标 | 验收标准 |
|------|----------|
| 验证家长是否愿意用「专用键盘」而非豆包 | 7 日留存 ≥ 15% |
| 验证付费意愿 | 免费额度用尽后转化率 ≥ 2% |
| 找到最高频子场景 | 单用户周均生成 ≥ 5 次 |

**不做（v1 排除）：** 家长对同学家长、语音输入、多轮对话建档、心理咨询、升学建议。

---

## 二、核心用户流程

```
微信收到消息
  → 长按复制
  → 切换到 Mami Key 键盘
  → 选择场景（对孩子 / 对老师）
  → 选择子场景 + 风格
  → （可选）补充一句背景
  → 生成 3 条候选回复
  → 点击一条 → 自动粘贴到输入框
  → 用户可微调后发送
```

## 三、功能模块

### 3.1 基础能力（P0，必须有）

| 功能 | 说明 |
|------|------|
| 输入法插件 | iOS / Android 至少先上一端；支持在微信等 App 内切换 |
| 剪贴板读取 | 切换键盘时自动读取已复制文本 |
| 场景选择 | 顶层 Tab：**对孩子** / **对老师** |
| 子场景选择 | 每类 5–6 个高频子场景（见第四节） |
| 风格选择 | 每场景 3–4 种风格（见第四节） |
| 多候选生成 | 每次生成 **3 条** 不同措辞的回复，供挑选 |
| 一键粘贴 | 选中后写入系统剪贴板或直接填入输入框 |
| 免费额度 | 每日 5 次免费生成；超出引导订阅 |
| 历史记录 | 本地保存最近 20 条生成记录（仅本机，不上传原文） |

### 3.2 体验增强（P1，MVP 后期或 v1.1）

| 功能 | 说明 |
|------|------|
| 背景补充 | 可选输入 1 句话上下文，如「孩子三年级」「老师刚批评过作业」 |
| 回复长度 | 短 / 中 两档（默认短，适合微信） |
| 敏感词过滤 | 拦截辱骂、威胁、甩锅、阴阳怪气等不推荐用语 |
| 再生成 | 对当前结果不满意，一键换一批 |
| 润色模式 | 用户先写草稿，AI 帮忙改得体（对老师场景优先） |

### 3.3 商业化（P0）

| 功能 | 说明 |
|------|------|
| 订阅 | 月卡 / 年卡；解锁无限生成 + 全部风格 |
| 试用 | 新用户 3 天不限次（可 A/B） |

### 3.4 明确不做（避免 scope 膨胀）

- 孩子年龄建档、成绩档案、长期关系分析  
- 代替家长做重大教育决策（转学、报班推荐）  
- 上传聊天截图 OCR（v2 再考虑）  
- 社区、UGC 话术广场

## 四、场景与风格矩阵

### 4.1 家长对孩子

| 子场景 | 说明 | 默认风格 |
|--------|------|----------|
| 催作业/学习 | 孩子拖延、不肯写 | 温和坚定 |
| 玩手机/屏幕时间 | 约定时间、收手机 | 温和坚定 |
| 顶嘴/发脾气 | 孩子情绪爆发 | 先共情再引导 |
| 表扬/鼓励 | 肯定努力而非结果 | 温暖鼓励 |
| 谈心/问学校 | 想了解在校情况 | 轻松平等 |
| 道歉/和好 | 家长冲动后修复关系 | 真诚简短 |

**可选风格：** 温和坚定 · 先共情再引导 · 温暖鼓励 · 轻松平等 · 真诚简短

**产品原则：** 不生成欺骗、威胁、羞辱、情感绑架类话术（如「白养你了」「不管你了」）。

---

### 4.2 家长对老师

| 子场景 | 说明 | 默认风格 |
|--------|------|----------|
| 请假 | 病假、事假 | 礼貌简洁 |
| 了解表现 | 问课堂、作业、习惯 | 礼貌简洁 |
| 反馈问题 | 作业太多、孩子不适应等 | 委婉合作 |
| 感谢/节日 | 教师节、学期末感谢 | 真诚得体 |
| 道歉 | 孩子犯错、家长疏忽 | 诚恳负责 |
| 约沟通 | 面谈、电话沟通 | 礼貌简洁 |

**可选风格：** 礼貌简洁 · 委婉合作 · 诚恳负责 · 真诚得体

**产品原则：** 不生成指责老师、攀比其他家长、越级投诉暗示、过度卑微讨好类话术。

## 六、技术 MVP 栈（建议）

| 层 | 方案 |
|----|------|
| 客户端 | 输入法 SDK + 极简配置页 App |
| 后端 | 单接口 `POST /generate`，场景参数 + 用户消息 |
| 模型 | 国内可用的大模型 API（通义/豆包/DeepSeek 等） |
| Prompt | 系统 Prompt 模板 + 场景变量（见 `prompts.md`） |
| 安全 | 输入输出敏感词过滤；不持久化用户聊天原文（或仅匿名统计） |

---

## 七、MVP 里程碑

| 阶段 | 周期 | 交付 |
|------|------|------|
| W1–2 | Prompt 与场景打磨 | 20 条黄金样例人工评测通过 |
| W3–4 | Android 输入法 + 后端 | 对内测用户可用 |
| W5–6 | 订阅 + 埋点 | 小范围妈妈群灰度（50–100 人） |
| W7–8 | 数据复盘 | 决定是否加 iOS / 扩展场景 |

---

## 八、成功指标（灰度期）

- **激活：** 安装后 24h 内完成首次生成 ≥ 60%  
- **留存：** 次日留存 ≥ 30%，7 日留存 ≥ 15%  
- **场景分布：** 记录对孩子 / 对老师占比，优化默认 Tab  
- **满意度：** 生成后「有用」点击率 ≥ 50%  
- **付费：** 免费额度用尽后订阅转化 ≥ 2%
